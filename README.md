## Before Publishing

1. Replace the hero placeholder with Barry's real approved photo or a licensed aviation image.
2. Update all canonical URLs if the final domain is not `barrybillcliff.com`.
3. Add real social/profile links to the `sameAs` schema array in `index.html`.
4. Add a contact email or media inquiry form if desired.
5. Upload to hosting and submit `sitemap.xml` in Google Search Console.

## Local Preview

From this folder:

```powershell
python -m http.server 8766 --bind 127.0.0.1 --directory .
```

Then open:

```text
http://127.0.0.1:8766/index.html
```

## Accuracy Guardrail

The First Circuit page uses safe language: petition allowed under 29 U.S.C. Sec. 660(a), permitting appellate review to proceed. It does not claim a merits reversal, exoneration, or a final no-ownership/no-control finding.
